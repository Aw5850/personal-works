构建工具是前端工程化的一些工具，方便维护、组织代码和团队协作，属于高级进阶知识。课程提到的工具了解其中一种及其基本原理即可（重点可学Gulp或FIS）。前端框架更新速度快，了解其基本原理，后续以不变应万变！
